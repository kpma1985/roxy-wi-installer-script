# Roxy-Wi_Installer
This script install the Roxy-Wi web interface for managing HA-Proxy servers in a Raspberry Pi.


# Install

```
git clone https://github.com/smilebasti/Roxy-Wi_Installer.git ~/Roxy-Wi_Installer
cd ~/Roxy-Wi_Installer
chmod +x install.sh
sudo ./install.sh
```
